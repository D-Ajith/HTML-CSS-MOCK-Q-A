CSS MOCK Q/A

1.What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to style the appearance of HTML elements on a web page.

2.What are the possible ways to apply CSS styles to HTML Document?

There are three possible ways to apply CSS styles to HTML Document there are: 1.Inline CSS. 2.Internal CSS. 3.External CSS.

3.Z-index

z-index is the CSS property that controls the stacking order of overlapping elements on a page.An element with a higher z-index value will appear in front of an element with a lower z-index value. The property is called “z-index”

4.Keyframes

The @keyframes rule specifies the animation code. The animation is created by gradually changing from one set of CSS styles to another.

5.Transforms

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move,Translate,skew, etc., elements.

6.Transistions

CSS transitions allows you to change property values smoothly, over a given duration. Mouse over the element below to see a CSS transition effect.

7.Pseudo classes

CSS pseudo-classes are used to add styles to selectors, but only when those selectors meet certain conditions. A pseudo class is expressed by adding a colon (:) after a selector in CSS, followed by a pseudo-class such as "hover", "focus", or "active", like this: a:hover { /* your style here */ }

8.Pseudo elements

A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element(s)

9.CSS selectors

A CSS selector selects the HTML element(s) you want to style. *We can divide CSS selectors into five categories: 1.Simple selectors (select elements based on name, id, class) 2.Combinator selectors (select elements based on a specific relationship between them) 3.Pseudo-class selectors (select elements based on a certain state) 4.Pseudo-elements selectors (select and style a part of an element) 5.Attribute selectors (select elements based on an attribute or attribute value)

10.CSS Combinators

A CSS selector can contain more than one simple selector. Between the simple selectors, we can include a combinator. There are four different combinators in CSS:
  • Descendant combinator (space), Child combinator (>), Next sibling combinator (+), Subsequent-sibling combinator (~).
